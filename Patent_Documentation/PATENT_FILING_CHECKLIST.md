### Quick Conversion Steps:

**Follow WORD_TO_PDF_QUICK_GUIDE.md for detailed instructions:**

1. Right-click each `.md` file → Open With → Microsoft Word
2. Apply IPO-compliant formatting:
   - Layout → Margins → Custom: **Top/Bottom/Left/Right: 2.5 cm**

   - Paper size: **A4**
3. File → Save As → PDF
   - Options → Check "PDF/A compliant"
4. Repeat for all 8 files
