## ✅ Step-by-Step Instructions

### For Each Document (Repeat 8 times):

#### 1. Open File in Word
- Right-click the `.md` file → Open With → Microsoft Word
- OR: Open Word first → File → Open → Browse to document

**Files to Convert:**
- ☐ PATENT_CLAIMS.md
- ☐ PATENT_DETAILED_DESCRIPTION.md

---

#### 2. Apply IPO-Compliant Formatting

**Page Setup:**
- Layout tab → Margins → Custom Margins
  - Top: **2.5 cm**
  - Bottom: **2.5 cm**
  - Left: **2.5 cm**
  - Right: **2.5 cm**
- Paper size: **A4** (210 × 297 mm)

**Font Settings:**
- Select All (Ctrl+A)
- Home tab → Font dropdown → **Times New Roman**
- Font size → **11 pt**
- Line spacing → **1.5 lines**
  - Home tab → Paragraph → Line Spacing → 1.5

**Optional - Page Numbers:**
- Insert tab → Page Number → Bottom of Page → Plain Number 3

---

#### 3. Save as PDF

**Method 1 - Save As:**
- File → Save As
- Choose location: `C:\Users\muham\OneDrive\Documents\#1_DMS`
- File name: Keep same name (e.g., `PATENT_CLAIMS`)
- Save as type: **PDF (*.pdf)**
- Options button → Check "PDF/A compliant"
- Click Save

**Method 2 - Export:**
- File → Export → Create PDF/XPS
- Location and filename as above
- Options → Check "PDF/A compliant"
- Click Publish

## ❓ Troubleshooting

**Problem:** Word doesn't open .md files  
**Solution:** Right-click → Open With → Choose App → Word → Always use

**Problem:** Formatting looks wrong  
**Solution:** Check View → Print Layout mode is enabled

**Problem:** PDF too large (>10 MB)  
**Solution:** In Word, File → Info → Compress Pictures → High Quality → Apply

**Problem:** Missing content in PDF  
**Solution:** Check original .md file in VS Code - scroll to end before converting

**Problem:** Can't find PDF/A option  
**Solution:** File → Save As → Tools → Save Options → Check "ISO 19005-1 compliant (PDF/A)"
